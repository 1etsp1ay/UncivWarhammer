# Unciv Warhammer

## WORK IN PROGRESS

An attempt to create a full mod (rulesets, map, graphics) of Warhammer for unciv.
Modmod of Fantasia mixed with previous Civ3/Civ4 Warhammer extensions materials.

Available nations:
* Reikland (The Empire)
* Bretonnia
* Kislev
* Sylvania
* Dwarfs
* Khemri
